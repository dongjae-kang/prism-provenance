# Korean Political Discourse Provenance Layer

한국 정치 YouTube에서 **어떤 주장이 어디에서 시작되어 어떻게 퍼지는지**를 매주 공개 대시보드로 정리합니다. 개별 사실을 검증하는 차원을 넘어, 정보가 확산되는 구조 자체를 드러냅니다.
